### Included in this Ionic React Template/UI
* Animated Sliding profile
* Dummy Profile Data
* Custom Tabs
* SASS Modules & Ionic CSS utilities

### To run

```javascript
npm install
ionic serve
```

Alternatively, you can add the iOS, Android platform and run natively.
